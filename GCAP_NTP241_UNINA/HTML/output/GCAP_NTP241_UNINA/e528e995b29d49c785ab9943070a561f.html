(FB-03) GPS Position [GNSS]

FeedbackElement

GCAP_NTP241_UNINA > GCAP_NTP241_UNINA > STPA Analysis > Control Structure > (C-05) Geolocation System > Feedback: (C-06) GNSS Source -> (C-05) Geolocation System > (FB-03) GPS Position [GNSS]

Receive coordinates information (from satellites to GPS).

Content

  • Causal Factors
  • (UCA-13) GNSS source fails to send coordinates to the GPS module (signal loss or transmission error).
  • (UCA-14) GNSS provides incorrect satellite data (e.g., due to spoofing, drift, or error in ephemeris data).
  • (UCA-15) GPS module misinterprets or misuses otherwise correct satellite data.
  • (UCA-16) GNSS signal is significantly delayed, affecting real-time position estimation.

Unsafe Control Actions

  • (UCA-13) GNSS source fails to send coordinates to the GPS module (signal loss or transmission error).
  • (UCA-14) GNSS provides incorrect satellite data (e.g., due to spoofing, drift, or error in ephemeris data).
  • (UCA-15) GPS module misinterprets or misuses otherwise correct satellite data.
  • (UCA-16) GNSS signal is significantly delayed, affecting real-time position estimation.

Supported Responsibilities

  • (R-10) Validate the integrity and accuracy of received GNSS data
  • (R-15) Exchange geolocation info with other components over a secured channel
