Receive Telemetry/sensor status (from Autopilot to Radio Control).

Content

  • Causal Factors

  • (UCA-06) The data sent is outdated, corrupted, or contains invalid sensor readings.
  • (UCA-07) Telemetry data is delayed, leading to outdated operator decisions.
  • (UCA-08) Incoherent sensor status is reported (e.g., IMU and GPS data mismatch).
