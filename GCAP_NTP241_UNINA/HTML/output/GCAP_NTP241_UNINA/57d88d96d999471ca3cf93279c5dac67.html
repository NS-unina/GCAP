(C-02) Radio Control

ControlEntity

GCAP_NTP241_UNINA > GCAP_NTP241_UNINA > STPA Analysis > Control Structure > (C-02) Radio Control

No description.

Content

  • The Radio Control unit shall verify signal integrity before transmitting commands to the Autopilot.
  • The Radio Control shall ensure that control commands are within valid operational limits before transmission.
  • The Radio Control shall trigger a failsafe mode if communication with the Operator is lost.
  • The Radio Control shall include command timestamps to detect delayed or stale commands.
  • The Radio Control shall periodically send heartbeat signals to verify bidirectional link availability.
  • The Radio Control shall only forward commands if telemetry (feedback F2) confirms system readiness.
  • (R-01) Transmit operator commands to the drone’s autopilot.
  • (R-02) Receive telemetry and sensor status from the Autopilot
  • (R-03) Exchange info with other components across a secure channel
