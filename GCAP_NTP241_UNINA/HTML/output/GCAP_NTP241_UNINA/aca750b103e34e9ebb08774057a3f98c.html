(CA-02) Flight Commands [Radio]

ControlAction

GCAP_NTP241_UNINA > GCAP_NTP241_UNINA > STPA Analysis > Control Structure > (C-02) Radio Control > Control: (C-02) Radio Control -> (C-03) Autopilot > (CA-02) Flight Commands [Radio]

Send flight commands to the Autopilot to influence motor behavior indirectly via high-level control inputs. 

Content

  • (UCA-01) The RC fails to send commands when user input is given (e.g., stick movement not transmitted).
  • (UCA-02) The RC transmits corrupted or inaccurate commands due to signal degradation or interface error.
  • (UCA-03) RC commands are sent during autonomous mode, causing interference.
  • (UCA-04) Commands are sent despite no operator input (e.g., noise interpreted as signal).
