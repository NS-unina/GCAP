System-Level Constraints

  • (SC-08) Information among drone components shall happen across secure channels

Losses

  • (L-02) Loss of Drone

Loss Scenarios

  • (LS-03) Tampering of Control Action CA2 via Command INjection Attack

Unsafe Control Actions

  • (UCA-04) Commands are sent despite no operator input (e.g., noise interpreted as signal).
  • (UCA-06) The data sent is outdated, corrupted, or contains invalid sensor readings.
  • (UCA-08) Incoherent sensor status is reported (e.g., IMU and GPS data mismatch).
  • (UCA-02) The RC transmits corrupted or inaccurate commands due to signal degradation or interface error.
  • (UCA-03) RC commands are sent during autonomous mode, causing interference.
